## 2024-03-07 - Accessible Rating Toggle Groups
**Learning:** When using custom button groups for rating selections (e.g., Skip, Timepass), treating them as a group of toggle buttons is essential for screen readers. Using `role="group"` on the container with an `aria-label`, adding `aria-pressed={selected === option}` to the buttons, and explicitly setting `type="button"` ensures correct reading and interaction. Additionally, hiding decorative emojis with `aria-hidden="true"` prevents redundant and confusing screen reader output (e.g., "thumbs down sign Skip"). Also, `textarea`s must have `aria-label` when no explicit `<label>` exists.
**Action:** Always apply `aria-pressed` to custom pill/toggle buttons to communicate state, group related buttons using `role="group"`, explicitly define `type="button"`, use `aria-hidden="true"` on decorative icons, and ensure inputs without labels have an `aria-label`.
